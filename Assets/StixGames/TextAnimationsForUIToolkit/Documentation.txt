To get started, add an AnimatedLabel in UI builder. Optionally you can create a TextAnimationSettings scriptable object
and assign it to your AnimatedLabel.

You can animate your text by adding tags to it, for example "Welcome to Text <wave>Animations</wave> for UI Toolkit".

For a more detailed guide, check out the documentation at https://docs.stixgames.com/text-animations
Also don't forget to look at the example scenes to see how to use various features in action.
